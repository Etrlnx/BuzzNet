# 🐝 YAMNet-Pest-Audio: Pest vs Beneficial Insect Classification

This repository contains a Kaggle notebook that leverages **Google’s YAMNet embeddings** and a **deep neural network** to classify insect audio recordings into two categories:  
- **Pest (0)**  
- **Beneficial (1)**  

The project integrates **TensorFlow Hub**, **Librosa**, and **Explainable AI (XAI)** techniques like **SHAP** and **Temporal Saliency** to ensure both accuracy and interpretability.

---

## 📂 Dataset
- Source: `InsectSound1000` (Kaggle dataset)  
- Total samples: **165,982** audio files  
- Balanced dataset after downsampling: **127,812** samples (63,906 per class)  

Labels are generated by mapping species names to categories (`pest` or `beneficial`) and stored in `insect_labels_binary.csv`.
